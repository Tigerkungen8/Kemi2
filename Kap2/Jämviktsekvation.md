# Definition
Gundläggande formeln $A + B \rightleftharpoons C + D \Rightarrow \frac{[C]\cdot[D]}{[A]\cdot[B]}=\mathbf{K}$ (konstant), när den grundläggande formeln är i jämvikt, alltså när kocentrationerna inte förändras mer.

\[] står för koncentrationen (enhet M för molar/dm³).

$\frac{[C]\cdot[D]}{[A]\cdot[B]}$ kallas även för *massverkans lag*/*Guldberg-Waages lag*.

$\mathbf{K}=\frac{Produkterna}{Reaktanterna}$ kan man också skriva.
