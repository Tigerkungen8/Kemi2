# Definition
Egentligen vad som helst som binder samman många [[Monomerer]], kan alltså vara en massa [[Kondensationsreaktion]]:er som skapar en lång [[Polymerer]], kallas då *polykondensation*.

Kallas också *polymerisation*.

## För [[Kolväten]]
[[Omättade]] moleyler som adderas till varandra (under högt tryck eller temperatur).

Exempel: $CH_2=CH_2 + CH_2=CH_2 \rightarrow CH_3-CH_2-CH=CH_2$. Etemolekyler adderas, och bildar 1-buten. Denna reaktion kan upprepas n antal gånger, som slutligen bildar *polyetenmolekyler*.
