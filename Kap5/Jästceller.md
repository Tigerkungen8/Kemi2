# Definition
[[Cellen]] som bygger uppp jäst (literally, jästsvampar är oftast encelliga) som endast använder [[Glykolysen]] för att få energi ut ur [[Kolhydrater]]. [[Pyrodruvsyra]]:an som bildas (och genererar 2 [[ATP]]-molekyler) omvandlas till etanol (två kolatomer) och koldioxid (en kolatom), vilket är därför man jäser saker för att bilda (drickbar) alkohol. Dessutom vid bakning, för att gasa upp degen så att den blir stor och fluffig.

*Jäsning* = mikroorganismer utvinner energi ur organiska föreningar genom att spjälka dessa anaerobt.
